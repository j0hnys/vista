~ gia na kanw ena local package prepei na exw
    - gia to kentriko composer.json
        - "*/*": "dev-master" sto require
        - to parakatw sto "repositories"
        ```
        "repositories": [
            {
                "type": "path",
                "url": "packages/*/*",
                "options": {
                    "symlink": true
                }
            }
        ],
        ```
        - to parakatw sto "autoload" -> "psr-4"
        ```
        "autoload": {
            "psr-4": {
                "App\\": "app/",
                "*\\*\\": "packages/*/*"
            },
        },
        ```
    - gia to composer.json toy package
        ```
        "extra": {
            "laravel": {
                "providers": [
                    "*\\*\\*ServiceProvider"
                ]
            }
        }
        ```
	
gia na exw t process.env.MIX_BASE_RELATIVE_URL prepei na valw to `MIX_BASE_RELATIVE_URL=/demo_omp_john/public` sto `.env` ths laravel


SUPER SOS!!
gia na kanw dyo diaforetika single page applications (client ennow) prepei:
    - na kanw diaforetikoys fakeloys `resources` k `public` gia front-end k back-end. p.x. `public`, `public_front`, `resources`, `resources_front`
    - na valw to
        ```
        //
        //dika moy ayta!!
        $app->bind('path.public', function() {
            return __DIR__;
        });
        //
        //
      sto index.php apo kathe public fakelo (gia t `public`, `public_front` sthn periptwsh moy)
    - na allaksw to webpack.mix.js ths laravel se:
        ```
        mix.setPublicPath(('../vista-framework/'));

        mix.js('resources/js/app.js', 'public/js')
        .sass('resources/sass/app.scss', 'public/css');

        mix.js('resources_front/js/app.js', 'public_front/js')
        .sass('resources_front/sass/app.scss', 'public_front/css');
        ```
    - etsi tha ginontai taytoxrona ola compile san na eixa mia efarmogh
